Franka Panda Visual Servoing

The Challenge

Implementing precise pick-and-place tasks using an industrial robotic arm in dynamic environments where object positions are not fixed.

The Solution

Developed a visual servoing pipeline in ROS 2 Humble. Integrated a camera with the Franka Panda manipulator to provide real-time feedback, enabling the arm to adjust its trajectory and grasp objects with high precision based on visual cues.

RTAB-Map SLAM & 3D Reconstruction

The Challenge

Generating high-fidelity 3D maps of indoor and outdoor environments using a mobile robot platform (Husky) with limited sensor range.

The Solution

Implemented RTAB-Map (Real-Time Appearance-Based Mapping) for SLAM and 3D reconstruction. Processed synchronized RGB-D and LiDAR data to create dense point clouds and navigable occupancy grids for autonomous exploration.

LiDAR-Inertial SLAM (ROS 2)

The Challenge

Achieving stable localization for robots moving at high speeds or over uneven terrain where standard wheel odometry fails.

The Solution

Integrated 3D LiDAR data with Inertial Measurement Unit (IMU) readings to perform robust SLAM. Optimized the fusion using a factor-graph-based approach, significantly reducing drift in feature-poor environments.

Leader-Follower Formation Control

The Challenge

Coordinating multiple differential-drive robots to maintain a specific geometric formation while following a moving leader.

The Solution

Implemented a leader-follower control strategy using Lyapunov-based stability analysis. Each follower robot maintains its relative distance and angle to the leader, adapting dynamically to changes in the leader's velocity and direction.
