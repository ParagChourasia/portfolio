Visual-Inertial SLAM Pipeline

The Challenge

Autonomous drones struggle with precise localization in environments where GPS is unavailable or unreliable, leading to drift in navigation.

The Solution

I developed a robust SLAM pipeline that tightly couples IMU data with stereo vision features using an Extended Kalman Filter (EKF). This ensures sub-centimeter accuracy even during aggressive maneuvers.

DEMO: Autonomous Navigation

Dynamic Path Planning

The Challenge

Navigating mobile robots in crowded spaces requires predicting human movement and planning paths that are both safe and efficient.

The Solution

Integrated RRT* with a Dynamic Window Approach (DWA) in a ROS 2 Nav2 stack. The system updates its local costmap at 20Hz, allowing the robot to weave through moving obstacles without stopping.

DEMO: Point Cloud Processing

LiDAR Object Detection

The Challenge

Extracting 3D bounding boxes from sparse LiDAR point clouds in real-time for autonomous driving applications.

The Solution

Implemented an end-to-end 3D detection pipeline using PointNet++. Optimized the inference on NVIDIA Jetson hardware using CUDA and TensorRT, achieving 15 FPS on a low-power edge device.
